Question-1: What are the different ways to select an element in the DOM?

we select the element basically 3 ways: 1.get element by Id method: where we set id at html and then we find out getElementById("idName"), 2.get elements by class name method: were we find out getElementsByClassName("className") then we find HTML collection its a array like object here we not apply forEach method. 3.we also find out query selector method: we we find any css class in query selector method: a.querySelectorByAll(".exampleClass") we get nodeList i.e array like object here we apply forEach method. b.querySelector(".exampleClass") we get first element.
