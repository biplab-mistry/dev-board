Question-3: What is event delegation in the DOM?

event delegation is the process which event is apply on the parent element except each child element, i.e if new child element create the event also apply due to apply event on parent element, it can prevent memory consume.

Question-4: What is event bubbling in the DOM?

event bubble is a process is that is, when an event apply on child element then first capture where the event is apply then it goes back to parent element.this process is called event bubble.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, we use document.createElement, then set its content, style, or attributes. To add it, attach it to a parent element using appendChild , To remove an element,we use remove() to delete it directly or removeChild() to remove it from its parent.
